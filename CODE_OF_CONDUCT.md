# Code of Conduct

## Our Commitment

District 76 contributors and maintainers are committed to a welcoming,
inclusive, and harassment-free community for everyone.

## Expected Behavior

- Be respectful in all interactions.
- Assume good intent and ask clarifying questions.
- Give actionable, constructive feedback.
- Focus discussion on ideas and code, not people.

## Unacceptable Behavior

- Harassment, intimidation, or discrimination.
- Personal attacks, insults, or derogatory language.
- Publishing private information without consent.
- Any conduct that would be inappropriate in a professional setting.

## Enforcement

Maintainers may remove, edit, or reject comments, commits, issues,
and contributions that violate this Code of Conduct.

## Reporting

If you experience or witness unacceptable behavior, open a private issue or
use your project-maintainer contact channel. Include as much context as
possible so maintainers can investigate.

## Scope

This Code of Conduct applies in project spaces and in public spaces when an
individual is representing this project or community.
