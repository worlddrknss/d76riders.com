# District 76

District 76 is a community-focused motorcycle platform built with Next.js.

## Development

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Run lint checks:

```bash
npm run lint
```

## Database Tasks

```bash
npm run db:generate
npm run db:migrate
npm run db:seed
npm run db:studio
```
