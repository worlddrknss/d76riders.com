# Security Policy

## Reporting a Vulnerability

If you discover a security issue, please report it privately to the project
maintainers first. Do not open a public issue for unpatched vulnerabilities.

When reporting, include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested remediation

## Response Process

Maintainers will acknowledge reports, investigate, and provide updates as time
permits. After a fix is available, coordinated public disclosure can follow.
